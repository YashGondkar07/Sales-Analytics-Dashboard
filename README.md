📊 Sales Analytics Dashboard
Overview

This project is an interactive Sales Analytics Dashboard built using Power BI. The dashboard analyzes sales performance, profit trends, regional performance, product performance, and customer segments to help businesses make data-driven decisions.

Objectives
Analyze overall sales and profit performance
Identify top-performing products
Compare sales across regions
Track monthly sales trends
Enable interactive filtering using slicers
